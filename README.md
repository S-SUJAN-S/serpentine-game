# 🐍 Serpentine — 3D Snake Game

A fast-paced 3D snake game built with Three.js and vanilla JavaScript.
No frameworks, no installs — runs entirely in the browser.

🎮 **[Play Now → serpentine-game.netlify.app](https://serpentine-game.netlify.app)**

## ✨ Features

- 🎯 3 difficulty levels — Easy, Medium, Hard
- 🏆 Persistent high scores per difficulty
- 📱 Mobile support — Joystick, Swipe and Mouse controls
- ✨ WebGL 3D graphics with particle effects
- 🎖️ Achievement system with milestone toasts
- 📊 Score history and session stats
- ⚡ Zero dependencies — pure HTML + JS

---

## 🕹️ Controls

| Input | Action |
|-------|--------|
| Mouse | Guide snake |
| WASD / Arrow Keys | Keyboard control |
| Space | Pause / Resume |
| R | Restart |
| T | Toggle walls |
| M | Switch control mode |
| ESC | Settings |
| ` (backtick) | Debug overlay |

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| [Three.js r128](https://threejs.org/) | 3D WebGL rendering |
| Vanilla JavaScript ES6+ | Game logic |
| HTML5 + CSS3 | Structure & styling |
| localStorage | Persistent high scores |

## 🚀 Run Locally

Just open `index.html` in any modern browser. No server needed.

```bash
git clone https://github.com/S-SUJAN-S/serpentine-game.git
cd serpentine-game
# Open index.html in your browser
```
